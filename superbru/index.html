SuperBru

Calculate the scoreline with the highest expected SuperBru points.

▶ How does this work?

This tool uses Expected Goals data from Understat to generate probabilities for different scorelines in Premier League games. In SuperBru, you score points by exactly or closely predicting the score of a game, as governed by these rules. A simple Python script calculates the expected points for each prediction and outputs the best choice. The way that the prediction home and away teams' xG maps onto the best prediction is shown in the plot below. Curiously, no draw other than 0-0 is ever the best Superbru prediction. This stems from the fact that draws have fewer scorelines that are considered 'close' by the SuperBru scoring system than wins.
